# Domekit — skill

Set installabile delle skill del **dome-method** (Domekit). Questo repo contiene
solo ciò che serve all'install: `skills/` (con i contratti e i database in
`skills/_shared/`) e l'indice `SKILLS.md`.

## Installazione

Richiede Node.js 18+:

```bash
npx skills add aitch00/domekit
npx skills update
```

Le skill atterrano nella cartella convenzionale dell'agente (`.claude/skills/`
per Claude Code, `.agents/skills/` per Cursor/Codex/Copilot).

Indice completo delle skill: [`SKILLS.md`](./SKILLS.md).
